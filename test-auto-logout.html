Auto-Logout Test Page

This is a simple test to verify the auto-logout functionality is working.

Open the browser console to see debug messages.

If you set auto-logout to 1 minute and don't interact with the page, you should be logged out after 1 minute.

Current Status:

  • Auto-logout functionality has been implemented in the position auth context
  • It loads settings from the database on user login
  • It sets up activity listeners (mouse, keyboard, touch, scroll)
  • It starts an inactivity timer based on the configured delay
  • When the timer expires, it automatically logs out the user

To test:

  1. Go to the System Settings page
  2. Set "Auto Logout Delay" to 1 minute
  3. Save the settings
  4. Navigate to any page in the application
  5. Don't interact with the page for 1 minute
  6. You should be automatically logged out and redirected to the login page
